remove介绍

在 C++ 中，remove 是一个算法函数，定义在 <algorithm> 头文件中。remove 并不会真正删除元素，而是将符合条件的元素移到容器的末尾，并返回一个指向新的逻辑结尾的迭代器。这样做是为了保持容器的连续性。

remove 函数通常与容器的修改函数（例如 erase）结合使用，以实际删除元素。这个组合通常用于从容器中删除特定值或元素。
